## Body (3–5 bullets max, OR a key code/command block, OR a comparison table)

- Agent notifications fire when Claude needs approval, so you stop watching a blinking cursor
- Rich input editor for long prompts; share inline code review comments back to the agent
- Attach selected snippets as context — no copy-paste gymnastics
- Share sessions with teammates for over-the-shoulder reviews

```bash
# one-time plugin install in any Warp pane
claude plugin marketplace add warpdotdev/claude-code-warp
claude plugin install warp@claude-code-warp
```
