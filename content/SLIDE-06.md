## Body (3–5 bullets max, OR a key code/command block, OR a comparison table)

- A plugin (obra/superpowers) that bundles 50+ skills covering the full dev loop
- Skills are markdown contracts the agent reads on demand — brainstorm, plan, test, ship
- Opinionated: brainstorm before code, write the failing test first, verify before claiming done
- Installable from the official plugin marketplace; one command per project

```bash
/plugin marketplace add obra/superpowers-marketplace
/plugin install superpowers@superpowers-marketplace
```
