## Body (3–5 bullets max, OR a key code/command block, OR a comparison table)

| Direction | What you do | What you get |
|---|---|---|
| Figma → Code | Reference a Figma frame URL in a prompt | Design-informed code generation with real tokens |
| Code → Figma | Capture a running UI from localhost or prod | Editable Figma frames on the canvas |
| Side-by-side | Compare multiple screen states | Catch inconsistencies without rewriting code |

- Bridges design-first and code-first teams against the same artefact
- Works with running UIs in localhost, staging, or production
- Pairs with the Devspiration design system in this very deck
